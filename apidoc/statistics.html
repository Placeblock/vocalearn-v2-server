Statistics

Each user has points for each vocable in both directions. With this part of the API you can store stats and receive stats of individual vocables.

Add Statistics Request

With this request you can add statistics to a vocable

HTTP-Request (POST)

/statistics/add

Parameter

"vocable" (The id of the Vocable you want to add statistics to)

"known" (Has the user known the vocable?)

"direction" (n_text -> f_text or f_text -> n_text ?)

Get Statistics Request

With this request you can get statistics for a vocable

HTTP-Request (GET)

/statistics/get

Parameter

"vocable" (The id of the vocable)

Receive

The statistics of the passed vocable

{
