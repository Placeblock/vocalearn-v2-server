Lesson

Lessons are a collection of vocables. They are used to learn Vocables. Example: Learn 10 Vocables from this lesson.


Create Lesson Request

With this request you can create new lessons

HTTP-Request (POST)

/lesson/create

Parameter

"name" (The name of the new lesson)

Receive

The lesson id

{

"id":id

}

Delete Lesson Request

With this request you can delete lessons

HTTP-Request (DELETE)

/lesson/delete

Parameter

"id" (The id of the lesson)

Lesson Metadata Request

This request responds with the metadata of the specified lesson, consisting of the name, the language and the creationdate

HTTP-Request (GET)

/lesson/metadata

Parameter

"id" (The id of the lesson)

Receive

The lesson metadata

{

"metadata":

{

"name": name,

"creation_date": creation_date

}

}

Lesson Content Request

With this request you can receive all of the vocables, which are inside of this lesson.

HTTP-Request (GET)

/lesson/content

Parameter

"id" (The id of the lesson)

Receive

The lesson content

{

"content":

{

vocableid:

{

"n_text":n_text,

"f_text":f_text,

"hint":hint

},

vocableid:

{

"n_text":n_text,

"f_text":f_text,

"hint":hint

}

}

}

User's Lessons with Metadata Request

Responds with all lessons of the user which is logged in with the additional metadata.

HTTP-Request (GET)

/lesson/metadatauser

Receive

The user's lessons with metadata

{

"lessons":

{

lessonid:

{

"name": name

"creation_date": creation_date

},

lessonid:

{

"name": name

"creation_date": creation_date

}

}

}

User's Lessons Request

Responds with all lessons of the user which is logged in.

HTTP-Request (GET)

/lesson/user

Receive

The user's lessons

{

"lessons":

[

lessonid,

lessonid

...

]

}

Search Lesson Request

With this request you can search trough all lessons in the database with regex.

HTTP-Request (GET)

/lesson/search

Parameter

"regex" (The regex to search)

Receive

The matched lessons

{

"lessons":

[

lessonid,

lessonid

...

]

}
