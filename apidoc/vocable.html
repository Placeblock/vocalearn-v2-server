Vocable

Each vocable consists of the n_text which is the first language text, the f_text which is the other language text and the hint which is something that could be usefull to remember t he vocable better.

Create Vocable Request

With this request you can create new vocables

HTTP-Request (POST)

/vocable/create

Parameter

"lesson" (The name of the lesson where the vocable should be put in)

"n_text" (Described above)

"f_text" (Described above)

"hint" (Described above)

Receive

The vocable id

{

"id":id

}

Delete Vocable Request

With this request you can delete vocables

HTTP-Request (DELETE)

/vocable/delete

Parameter

"id" (The id of the vocable)

Edit Vocable Request

With this request you can edit vocables

HTTP-Request (PUT)

/vocable/edit

Parameter

"id" (The id of the vocable)

"lesson" (The new lesson)

"n_text" (Described above)

"f_text" (Described above)

"hint" (Described above)

Search Vocable Request

With this request you can search trough all vocables in the database with regex. (Usefull for dictionary)

HTTP-Request (GET)

/vocable/search

Parameter

"regex" (The regex to search)

Receive

The matched vocables

{

"vocables":

[

vocableid,

vocableid
