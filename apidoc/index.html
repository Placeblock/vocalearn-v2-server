General

This is the API Documentation for the Vocalearn API. With Vocalearn you can learn vocables, manage lessons, create selections, show your statistics and more. You can use this API to comunicate with the Vocalearn Database.



Attention:

Each Request could respond with an error! A good example is trying to get a lesson's metadata without specifying your API Token:

HTTP-Request (GET)

/lesson/metadata

Parameter

"id" (The lesson id)

Receive

An error response

{

"error":

{

"httpcode": 401

"errorcode": 1001

"message": "Unauthorized! Please add a token!"

}

}

The structure of error responses are always the same.
