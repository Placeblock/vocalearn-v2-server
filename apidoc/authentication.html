Authentication

The API is Tokenbased. That means, to use Vocalearns API you have to login with user credentials, and if they are correct you will receive a token which you can save somewhere in your App. In each API request you should add a "token" parameter which contains the token you received earlier.


Login Request

In the Login Request you shouldn't pass a token, normally because you don't have one. This is how the request should look like:

HTTP-Request

/token/login

Parameter

"email" (The email of the user)

"password" (The password of the user)

Receive

The API Token

{

"token":token

}

Logout Request

When you login with the users credentials, a token is generated and stored in the database. If you logout with this request, the token gets deleted.

HTTP-Request

/token/logout

Parameter

"token" (The token you want to logout)
