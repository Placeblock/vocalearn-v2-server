Selection

A selection contains multiple lessons. Example: An user created 5 lessons for english subject. Now there is an exam and the user wants to learn all the five lessons for it. Then the user adds all 5 lessons to a created selection and learns just this selection!

Create Selection Request

With this request you can create new selection

HTTP-Request (POST)

/selection/create

Parameter

"name" (The name of the new selection)

Receive

The selection id

{

"id":id

}

Delete Selection Request

With this request you can delete selections

HTTP-Request (DELETE)

/selection/delete

Parameter

"id" (The id of the selection)

Add Lesson Request

With this request you can add a lesson to a selection

HTTP-Request (POST)

/selection/addlesson

Parameter

"id" (The id of the selection)

"lesson" (The id of the lesson)

Remove Lesson Request

With this request you can remove a lesson from a selection

HTTP-Request (DELETE)

/selection/removelesson

Parameter

"id" (The id of the selection)

"lesson" (The id of the lesson)

Selection Metadata Request

This request responds with the metadata of the specified selection, consisting of the name and the creationdate

HTTP-Request (GET)

/selection/metadata

Parameter

"id" (The id of the selection)

Receive

The selection metadata

{

"metadata":

{

"name": name,

"creation_date": creation_date

}

}

Selection Content Request

With this request you can receive all of the vocables, which are inside of this selection.

HTTP-Request (GET)

/selection/content

Parameter

"id" (The id of the selection)

Receive

The selection content

{

"content":

[

lessonid,

lessonid

...

]

}

User's Selections Request

Responds with all selections of the user which is logged in.

HTTP-Request (GET)

/selection/user

Receive

The user's selections

{

"selections":

[

selectionid,

selectionid

...

]

}
